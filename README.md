# 🚀 Vidya Setu – AI-Powered Inclusive Learning Platform

**"Smart. Inclusive. Multilingual. Education Reimagined."**

Vidya Setu (विद्या सेतु – *Bridge of Knowledge*) is an AI-powered educational platform that delivers personalized, multilingual, and accessible learning for every Indian student.

---

## 🌟 Features

- 🎯 **Adaptive Quiz Engine** – Difficulty adjusts based on performance  
- 🌐 **Multilingual Support** – 10 Indian languages  
- 🧠 **RAG Chatbot** – Upload PDFs and get context-aware explanations  
- ♿ **Accessibility Modes** – ADHD, dyslexia, visual impairment  
- 🔊 **Text-to-Speech** – Natural Indian voice output  
- 📊 **Weak Topic Detection** – Focus on improvement areas
